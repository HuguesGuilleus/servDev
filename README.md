# HTTP Server for developping project

A little programe to run a HTTP file server.

## Installation

Copy release binary into your PATH or build yourself (`go build`).

## Usage

| Option | Description              |
| -----: | :----------------------- |
|     -a | Listen address           |
|     -c | TLS certificate          |
|     -k | TLS key                  |
|     -s | Listen address for HTTPS |
